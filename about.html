Hi I'm Isaac Mercer, a 15 year old student from Auckland, New Zealand. I have an interest in technology, both the software and the hardware, I love to programme in my spare time and create projects for myself and others.

I mainly programme in HTML, CSS and PHP, I also do a bit of Apple's Swift. For developing web apps I use the idea of creating the interface in HTML and CSS and then display it consistently across the devices with a WebView. My app SafeDriver uses this idea. The only code running on the device itself is device specific hardware functions. The actual UI served is an HTML page displayed in a WebView which appears to be running natively by using elements present in mobile app designs.
